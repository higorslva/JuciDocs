Introdução Last updated: 10 de fevereiro de 2025

O ChatBot JUCI é uma solução projetada para utilizar modelos LLMs de código aberto, como Gemma2 ou outros modelos compatíveis com a biblioteca Ollama, para fornecer respostas precisas e baseadas em técnicas de Retrieval-Augmented Generation (RAG).

Esta versão OpenSource requer a configuração de dois componentes principais:

  • Cliente: Responsável por executar o backend (Python, Ollama e Qdrant).
  • Servidor: Responsável pelo frontend em PHP.

    • Configuração do Servidor

    Para o Servidor, podemos utilizar as seguintes configurações:

    Criar um Ambiente Virtual Python: python -m venv juci

    Ativar o ambiente virtual:

    • Windows: juci\Scripts\activate
    • Linux/Mac: source juci/bin/activate

    Instalar as dependências no ambiente virtual: pip install -r requirements.txt

    Note

    A JUCI foi testada em um ambiente Linux com Python 3.12.x. Teoricamente é possível rodar em versões mais recentes, mas não garantimos.

    Recursos

    A assistente virtual Juci (inspirado na palavra judiciário) foi desenvolvida para facilitar a interação e aproximar o cidadão aos serviços do Tribunal de Justiça do Amapá. Com uma interface amigável e simples, a Juci responde a perguntas frequentes, consulta informações processuais e orienta os usuários em tempo real.

    O que o Chatbot pode fazer

    O chatbot Juci oferece as seguintes funcionalidades:

  • Consulta processual: Realiza consultas no Tucujuris e futuramente no Processo Judicial eletrônico (PJe);
  • Perguntas frequentes: Responde a perguntas comuns relacionadas ao sistema judicial, procedimentos legais e orientações gerais, além de informações pertinentes do tribunal, tais como localização de comarcas e/ou secretarias, número de contato, email e balcão virtual;
  • Orientações em tempo real: Ajuda o usuário a navegar pelo sistema e fornece informações sobre como realizar tarefas específicas, como acompanhamento de um processo ou geração de documentos;
  • Acompanhamento de contexto: O chatbot pode manter o contexto de uma conversa ativa para melhorar a experiência do usuário durante uma sessão de consulta.

    • O que o Chatbot não pode fazer

    Apesar de suas diversas funcionalidades, o chatbot JUCI possui algumas limitações:

  • Respostas específicas a perguntas jurídicas complexas: O chatbot não substitui um advogado e não oferece aconselhamento jurídico. Ele apenas fornece respostas com base em dados disponíveis.
  • Processamento de informações confidenciais: Não trata ou lida com dados sigilosos, confidenciais ou privados fora do escopo permitido para consultas públicas.
  • Consultas a bancos de dados externos: Ele não realiza consultas em tempo real a outros sistemas fora do banco de dados do sistema Tucujuris.
  • Salvamento de informações pessoais: Não armazena dados fornecidos pelo usuário após o término da sessão de chat. Sendo assim, cada acesso da página, independente de ser o mesmo usuário ou não, o chat será reiniciado.
  • Consulta de dados pessoais: O chatbot NÃO consulta dados pessoais e/ou públicos de servidores do Tribunal de Justiça ou de qualquer outro órgão público e privado.

    • Tratamento de Dados

    Coleta de Dados

    O chatbot coleta apenas as informações necessárias para realizar as consultas solicitadas, como:

  • Número do processo;
  • Nome das partes envolvidas;
  • Histórico de conversa por sessão, para contexto e conversação contínua.

    • Privacidade e Segurança

  • Consentimento: Ao utilizar o chatbot, o usuário consente que as informações fornecidas durante a sessão sejam processadas temporariamente para a finalidade exclusiva da geração de resposta.
  • Armazenamento de Dados: NÃO HÁ nenhum tipo de armazenamento persistente de dados fornecidos pelos usuários na base de dados própria da Juci ou do TJAP. Todas as consultas são processadas em tempo real, e os dados são descartados imediatamente após o encerramento da sessão.

    • Não Armazenamento de Informações

    Nenhuma informação pessoal é armazenada após a sessão. Isso inclui:

  • Informações de login e/ou identificação pessoal;
  • Histórico de conversas;
  • Dados pessoais fornecidos para consulta;

    • Ao final de cada sessão, todas as informações são automaticamente removidas do sistema.

    Funcionamento da Coleta de Informações

    Interação com o Chatbot

    O usuário pode iniciar uma conversa fornecendo informações de consulta, como o número do processo, ou fazendo alguma pergunta sobre o Tribunal de Justiça do Amapá ou relacionado à área. O chatbot realiza a consulta com base nas informações fornecidas e retorna os resultados em tempo real.

    Temporariedade dos Dados

    Os dados são processados apenas durante a interação ativa. Assim que a sessão de chat é encerrada, todos os dados são excluídos do sistema, sem qualquer tipo de armazenamento ou processamento posterior.

    Nota: Este chatbot foi desenvolvido para otimizar o atendimento ao público em consultas processuais simples e dúvidas frequentes. Para questões mais complexas ou que envolvem dados confidenciais, recomendamos o contato direto com um advogado ou o órgão responsável.

    ChatBot JUCI

    O que o chatbot JUCI faz?

    O chatbot JUCI foi desenvolvido para facilitar consultas processuais e responder dúvidas do público geral. Ele permite que os usuários consultem o andamento de processos jurídicos e obtenham resumos de maneira simples e acessível. Ele também responde a perguntas gerais baseadas em um dataset específico, tornando o sistema prático para cidadãos sem conhecimentos jurídicos.

    O que o chatbot JUCI não faz?

    O chatbot JUCI não oferece consultoria jurídica, não toma decisões legais e não substitui o aconselhamento de advogados. Além disso, ele não acessa processos sigilosos, não pode modificar informações processuais e não consulta dados pessoais e/ou públicos de servidores do Tribunal de Justiça.

    Quais dados o chatbot JUCI utiliza?

    O chatbot utiliza informações públicas extraídas do portal web do TJAP e informações públicas de sistemas judiciais, como a API do Tucujuris. O foco principal é fornecer resumos de processos e informações relevantes para o cidadão.

    Onde os dados vão após a consulta?

    Os dados processuais consultados são armazenados temporariamente para garantir que o usuário possa fazer perguntas adicionais sobre o mesmo processo sem perder o contexto. No entanto, o chatbot não mantém esses dados de forma permanente a menos que o projeto inclua funcionalidades de histórico para uso autorizado (não implementado na v1.0).

    Como os dados são processados?

    Os dados processuais são acessados de APIs judiciais, como o Tucujuris, e processados usando técnicas de Processamento de Linguagem Natural (NLP) para gerar resumos compreensíveis. Esses resumos são feitos para que cidadãos sem conhecimento jurídico profundo possam entender de maneira simplificada o andamento do seu processo.

    Como são garantidos a privacidade e a segurança dos dados?

    O chatbot respeita as diretrizes de privacidade ao utilizar apenas dados públicos. Dessa forma, qualquer dado sensível é considerado inacessível pelo chatbot. É importante salientar que a segurança e privacidade desses dados estão sob responsabilidade exclusiva do sistema Tucujuris, pois o chatbot apenas acessa como um cliente a base de dados do Tucujuris.

    O chatbot aprende com as interações?

    O chatbot Juci utiliza uma arquitetura que pode se beneficiar do aprendizado de máquina para melhorar a precisão das respostas, mas isso depende da configuração do sistema. No momento, ele processa as consultas com base em modelos já treinados, e não armazena informações sobre o usuário, a menos que seja explicitamente configurado para isso (não implementado na versão atual).

    Quais são as limitações do chatbot?

    O chatbot JUCI depende de dados de externos, vindo do Tucujuris. Se a API fornecedora (isto é, a conexão do chatbot com o Tucujuris) estiver fora do ar, isso pode afetar as respostas. Além disso, a qualidade das respostas depende da completude dos dados fornecidos pelo sistema consultado.

    Posso integrar o chatbot a outros sistemas?

    Sim, o chatbot JUCI pode ser integrado a outros sistemas, desde que as APIs e serviços estejam disponíveis e compatíveis com a arquitetura do chatbot.

    Quem pode utilizar o chatbot?

    O chatbot foi projetado para ser utilizado tanto por cidadãos comuns que desejam acessar seus processos de forma simples quanto por advogados que necessitam de um assistente digital para acompanhar o andamento processual.
